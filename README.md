# Unity-ScreenSpaceShadow
A simple Screen Space Shadow effect for Untiy based on screen space. Not optimized, only for learning purpose.  
为Unity实现的简单的屏幕空间阴影。
基于Unity2017.1、DirectX。未在其他版本、平台测试。  
未进行优化，仅做学习目的。
